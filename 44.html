44 3n + 1 Problem



3n + 1 問題（The 3n + 1 Problem）

在電腦科學中，問題通常會被分類到不同的類別（例如 NP、不可解問題、遞迴問題等）。本題中，你將分析一個演算法的性質，而目前尚未知道它對所有可能的輸入是否都屬於哪一類。

請考慮以下演算法：
1. 輸入整數 n
2. 輸出 n
3. 如果 n = 1，則停止（STOP）
4. 如果 n 是奇數，則令 n = 3n + 1
5. 否則（n 是偶數），令 n = n / 2
6. 回到步驟 2

例如，當輸入為 22 時，會輸出以下數列：
22 → 11 → 34 → 17 → 52 → 26 → 13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1

關鍵概念：

目前有一個猜想：對於任何正整數作為輸入，這個演算法最後都會停止（也就是最終會輸出 1）。

儘管這個演算法非常簡單，但至今仍未證明這個猜想是否正確。不過，目前已經驗證，所有滿足 0 < n < 1,000,000 的整數，都會使此演算法停止。

對於一個輸入 n，從開始輸出到最後輸出 1（包含 1）為止，總共輸出的數字個數稱為 cycle-length（循環長度）。

例如上面的例子，22 的 cycle-length 為 16。

【你的任務】

給定兩個整數 i 和 j，求出 介於 i 和 j（包含兩端）所有整數之中，最大的 cycle-length。

特別要求：
1. 輸出的 i 和 j 必須保持與輸入時相同的順序
2. 輸出三個數字：i、j 和最大 cycle-length，使用空白隔開

輸入說明 輸出說明
輸入包含多組測試資料，每行包含一組兩個整數 i 和 j

所有整數皆滿足：0 < i, j < 10000

當 i 和 j 同時為 0 時，表示輸入結束		 對於每組輸入，輸出一行：
i j 最大_cycle_length

注意：輸出的 i 和 j 必須保持與輸入時相同的順序

          # 這個函數應該：
          # 1. 輸入一個正整數 n
          # 2. 根據演算法反覆計算 n 的值

          cache = {1: 1}

          def CalculateCycleLength(n):
              # 使用快取表避免重複計算 3n+1 序列
              seq = []
              while n not in cache:
                  seq.append(n)
                  if n % 2 == 1:  # 如果 n 是奇數
                      n = 3 * n + 1
                  else:  # 如果 n 是偶數
                      n = n // 2

              length = cache[n]
              for x in reversed(seq):
                  length += 1
                  cache[x] = length
              return cache[seq[0]] if seq else length

副程式：CalculateCycleLength() - 計算單一整數的 cycle-length 

          cache = {1: 1}

          def CalculateCycleLength(n):
              # 使用快取表避免重複計算 3n+1 序列
              seq = []
              while n not in cache:
                  seq.append(n)
                  if n % 2 == 1:  # 檢查 n 是否為奇數（n 除以 2 的餘數是否為 1）
                      n = 3 * n + 1  # 如果是奇數，執行 n = 3n + 1
                  else:  # 否則 n 是偶數
                      n = n // 2  # 執行 n = n / 2（使用整數除法）

              length = cache[n]
              for x in reversed(seq):
                  length += 1
                  cache[x] = length
              return cache[seq[0]] if seq else length
副程式：FindMaxCycleLength() - 找出範圍內的最大 cycle-length 

          def FindMaxCycleLength(i, j):
              # 確定範圍的最小值和最大值
              min_val = min(i, j)
              max_val = max(i, j)
              
              # 初始化最大值為 0
              max_cycle = 0
              
              # 遍歷範圍內的每個整數
              for n in range(min_val, max_val + 1):
                  # 計算當前整數的 cycle-length
                  current_cycle = CalculateCycleLength(n)
                  # 如果當前的 cycle-length 大於目前的最大值，更新最大值
                  if current_cycle > max_cycle:
                      max_cycle = current_cycle
              
              return max_cycle
