46. 翻轉排序


在電腦科學中，排序是一個非常重要的環節。如果資料已經預先排序好，幾乎所有的問題都能被有效率地解決。

在這道題目中，我們將探討一種新的排序方法。在這種方法裡，你只能使用一種操作——「翻轉（Flip）」，也就是交換兩個相鄰的元素。稍微思考一下就會發現，用這種方式一定能將任何一組數字排序完成。

題目會給定一組整數。現在，我們想要用上述的方法，將這些數字由小到大（遞增）排序。你必須找出最少需要幾次翻轉操作。

例如：要將 1 2 3 排序不需要任何翻轉操作（0 次）；而要將 2 3 1 排序，則至少需要 2 次翻轉操作。

輸入與輸出說明


輸入說明：

  輸入包含多組測試資料。
  每組測試資料開頭會給定一個正整數 $N$（$N \le 1000$）。
  接下來的幾行會包含 $N$ 個整數。
  輸入會持續進行，直到檔案結束（EOF）為止。

輸出說明：

  對於每組測試資料，請印出以下格式的訊息：
    Minimum exchange operations : M

  其中 M 是完成排序所需的最少翻轉操作次數。每一組測資的輸出必須獨立成一行。

輸入輸出範例 Input Output
範例測資 3
1 2 3
3
2 3 1 		Minimum exchange operations : 0
Minimum exchange operations : 2

Hints



                    # Step 1: 準備資料
                    # 把所有輸入讀進來，切分後變成一個裝滿數字字串的「牌堆」(data)。

                    # Step 2: 逐一處理 (只要牌堆還有牌就繼續)
                    # 使用 pop(0) 從牌堆最前面「抽出」第一個數字，這就是陣列大小 N。
                    # 準備一個空的陣列 a。
                    # 跑 N 次迴圈，連續從牌堆抽出 N 個數字放進陣列 a 裡面。

                    # Step 3: 計算逆序對 (Inversions)
                    # 準備一個計數器 (ans)。
                    # 使用雙層迴圈，拿出陣列裡的每一個數字 (第 i 個)，
                    # 並且跟它「後面」的所有數字 (第 j 個) 做比較。
                    # 如果發現「前面的數字大於後面的數字」，計數器就加 1。

                    # Step 4: 輸出結果

解答

主程式 

                    import sys

                    def solve():
                        # 讀取所有輸入內容，變成一個由左到右排好的「數字牌堆」
                        data = sys.stdin.read().split()

                        # 只要牌堆裡面還有數字，就繼續處理下一組測資
                        while len(data) > 0:
                            # 抽出最前面的 1 個數字，當作陣列大小 N
                            n = int(data.pop(0))
                            
                            a = []
                            # 跑 N 次迴圈，抽出接下來的 N 個數字裝進陣列
                            for _ in range(n):
                                a.append(int(data.pop(0)))
                                
                            ans = 0
                            # 雙層迴圈計算逆序對
                            for i in range(n):
                                for j in range(i + 1, n):
                                    # 如果前面的數字大於後面的數字 (轉義 >)
                                    if a[i] > a[j]:
                                        ans += 1
                                        
                            # 輸出結果
                            print(f"Minimum exchange operations : {ans}")

                    if __name__ == '__main__':
                        solve()
