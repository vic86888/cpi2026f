46. 翻轉排序


在電腦科學中，排序是一個非常重要的環節。如果資料已經預先排序好，幾乎所有的問題都能被有效率地解決。

在這道題目中，我們將探討一種新的排序方法。在這種方法裡，你只能使用一種操作——「翻轉（Flip）」，也就是交換兩個相鄰的元素。稍微思考一下就會發現，用這種方式一定能將任何一組數字排序完成。

題目會給定一組整數。現在，我們想要用上述的方法，將這些數字由小到大（遞增）排序。你必須找出最少需要幾次翻轉操作。

例如：要將 1 2 3 排序不需要任何翻轉操作（0 次）；而要將 2 3 1 排序，則至少需要 2 次翻轉操作。

輸入與輸出說明


輸入說明：

  輸入包含多組測試資料。
  每組測試資料開頭會給定一個正整數 $N$（$N \le 1000$）。
  接下來的幾行會包含 $N$ 個整數。
  輸入會持續進行，直到檔案結束（EOF）為止。

輸出說明：

  對於每組測試資料，請印出以下格式的訊息：
    Minimum exchange operations : M

  其中 M 是完成排序所需的最少翻轉操作次數。每一組測資的輸出必須獨立成一行。

輸入輸出範例 Input Output
範例測資 3
1 2 3
3
2 3 1 		Minimum exchange operations : 0
Minimum exchange operations : 2

Hints


                    # Step 1: 讀取陣列大小 N
                    # 讀取輸入的第一行，並轉換為整數 N。

                    # Step 2: 收集陣列資料
                    # 準備一個空的陣列 a。
                    # 因為測資的數字可能全部寫在同一行，也可能分好幾行寫，
                    # 所以使用迴圈持續讀取，直到陣列 a 裡面剛好裝滿 N 個數字為止。

                    # Step 3: 計算逆序對 (Inversions)
                    # 準備一個計數器 (ans)。
                    # 使用雙層迴圈，拿出陣列裡的每一個數字 (第 i 個)，
                    # 並且跟它「後面」的所有數字 (第 j 個) 做比較。
                    # 如果發現「前面的數字大於後面的數字」，計數器就加 1。

                    # Step 4: 輸出結果
                    # 印出最終計數器的數字，這就是最少需要的翻轉次數。

字串處理


在手動輸入測資時，有時會不小心多按空白鍵或換行，
這時 strip() 和 split() 就能幫我們把雜質過濾掉！ 


                    # 1. strip()：脫掉頭尾的多餘空白與換行符號
                    text = "   123   \n"
                    clean_text = text.strip() 
                    print(clean_text)  # 結果變成乾淨的 "123"

                    # 2. split()：把字串當作西瓜，遇到空白就切一刀，變成陣列
                    line = "2   3  1"
                    pieces = line.split()
                    print(pieces)      # 結果變成 ['2', '3', '1']
                    # 注意：它會聰明地無視連續的空白！

解答

主程式 

					def solve():
						# 1. 讀取陣列大小 N
						line = input().strip() # 讀取一行並去除頭尾多餘的空白
						if not line:
							return # 如果不小心直接按了 Enter，就結束程式
						n = int(line)
						
						a = []
						# 2. 持續讀取數字，直到陣列 a 裡面剛好裝滿 n 個數字為止
						while len(a) < n:
							elements = input().split() # 把輸入的一行文字用空格切開
							for x in elements:
								a.append(int(x)) # 把切開的數字一個一個轉成整數並放入陣列
								
						# 3. 開始計算逆序對
						ans = 0
						for i in range(n):
							# 內層迴圈從 i 的下一個位置開始找
							for j in range(i + 1, n):
								# 若前面的數字大於後面的數字(大於符號需寫成 >)，翻轉次數就加一
								if a[i] > a[j]:
									ans += 1
									
						# 4. 輸出結果
						print(f"Minimum exchange operations : {ans}")

					if __name__ == '__main__':
						solve()
