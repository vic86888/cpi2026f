48 What is the Median



What is the Median？（中位數是什麼？）

中位數（Median） 在統計學中扮演著重要的角色。

根據定義，中位數是能夠將一組資料分成兩個大小相等部分的數值。

例如，有五個數：{1, 3, 6, 2, 7}
排序後為：{1, 2, 3, 6, 7}

其中 3 就是中位數，因為它左右兩邊各有兩個數：
左邊：{1, 2}
右邊：{6, 7}

關鍵概念：

若資料個數為偶數，例如：{1, 3, 6, 2, 7, 8}
排序後為：{1, 2, 3, 6, 7, 8}

由於沒有單一一個數能將資料平分，因此中位數定義為中間兩個數的平均值：
(3 + 6) / 2 = 4.5

但本題只需要輸出整數部分，不保留小數。
因此，本題中的中位數為：4

【你的任務】
每讀入一個整數，就立即輸出目前所有已讀入數字的中位數。
也就是說：讀入第 1 個數後輸出一次、讀入第 2 個數後再輸出一次…… 每新增一個數，都要重新計算並輸出目前的中位數。

          # 提示 1：寫一個函數 CalculateMedian(numbers)
          # 這個函數應該：
          # 1. 輸入一個整數列表 numbers
          # 2. 將列表排序
          # 3. 計算中位數
          # 4. 回傳中位數（整數部分）
          
          def CalculateMedian(numbers):
              # 先將數字排序
              sorted_nums = sorted(numbers)
              n = len(sorted_nums)
              
              # 如果個數為奇數
              if n % 2 == 1:
                  return sorted_nums[n // 2]  # 回傳中間的數
              else:
                  # 如果個數為偶數，取中間兩個數的平均值
                  middle1 = sorted_nums[n // 2 - 1]
                  middle2 = sorted_nums[n // 2]
                  return (middle1 + middle2) // 2  # 整數除法（無條件捨去）

          def CalculateMedian(numbers):
              # 先將所有數字從小到大排序
              sorted_nums = sorted(numbers)
              n = len(sorted_nums)
              
              # 如果個數為奇數
              if n % 2 == 1:
                  # 中位數就是中間的那個數
                  # n // 2 會找到正中間的索引
                  return sorted_nums[n // 2]
              else:
                  # 如果個數為偶數，中位數是中間兩個數的平均值

                  middle2 = sorted_nums[n // 2]      # 第二個中間數
                  # 使用整數除法 // 來無條件捨去小數部分
                  return (middle1 + middle2) // 2
