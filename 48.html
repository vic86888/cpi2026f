48 What's the median



What is the median？（中位數是什麼？）

中位數（Median）在統計學中扮演著重要的角色。

根據定義，中位數是能夠將一組資料分成兩個大小相等部分的數值。

例如，有五個數：{1, 3, 6, 2, 7}
排序後為：{1, 2, 3, 6, 7}

其中 3 就是中位數，因為它左右兩邊各有兩個數：
左邊：{1, 2}
右邊：{6, 7}

關鍵概念：

若資料個數為偶數，例如：{1, 3, 6, 2, 7, 8}
排序後為：{1, 2, 3, 6, 7, 8}

這時候沒有單一一個數可以剛好把資料平分，所以我們取中間兩個數的平均值：
(3 + 6) / 2 = 4.5

但這道題只要輸出整數部分，不要小數。
所以答案會是：4

【你的任務】
每讀入一個整數，就要把它加入目前的數字串列，並立即輸出目前所有數字的中位數。
也就是說：讀入第 1 個數後輸出一次、讀入第 2 個數後再輸出一次……

          def CalculateMedian(numbers):
              numbers.sort()  # 先把數字排好
              length = len(numbers)

              if length % 2 == 1:
                  return numbers[length // 2]
              else:
                  left = numbers[length // 2 - 1]
                  right = numbers[length // 2]
                  return (left + right) // 2

副程式：CalculateMedian() - 計算中位數 

          def CalculateMedian(numbers):
              # 先把所有數字從小到大排好
              numbers.sort()
              length = len(numbers)

              # 如果目前有奇數個數字
              if length % 2 == 1:
                  # 中位數就是最中間那一個
                  return numbers[length // 2]
              else:
                  # 如果是偶數個，就取中間兩個數字的平均
                  left = numbers[length // 2 - 1]
                  right = numbers[length // 2]
                  # 題目要求只要整數部分，所以用 //
                  return (left + right) // 2
主程式：main() - 讀入資料並輸出中位數

          def CalculateMedian(numbers):
