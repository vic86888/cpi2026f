48 What's the median



What's the median？

中位數（Median）在統計學中扮演重要的角色。

依照定義，中位數是將一組資料分成左右兩半的數值。

在這個問題中，你需要在每讀入一個整數後，立即求出目前所有已讀入數字的中位數。

例如，目前已有數列：
> {1, 3, 6, 2, 7}

排序後變成：
> {1, 2, 3, 6, 7}

其中位數是 3，因為 3 左右各有兩個數字：
- 左邊：{1, 2}
- 右邊：{6, 7}


如果目前共有偶數個數字，例如：

> {1, 3, 6, 2, 7, 8}

排序後變成：
> {1, 2, 3, 6, 7, 8}

這時候沒有一個數字可以剛好把資料分成兩半，因此我們取中間兩個數字的平均值：
> (3 + 6) / 2 = 4.5

但本題只需要輸出整數部分，不需要小數部分。

因此：
> 中位數輸出為 4

提示 2：計算中位數的想法


                    def CalculateMedian(numbers):
                        numbers.sort()  # 先把數字從小到大排序
                        length = len(numbers)

                        if length % 2 == 1:
                            return numbers[length // 2]
                        else:
                            left = numbers[length // 2 - 1]
                            right = numbers[length // 2]
                            # 只要整數部分，無條件捨去
                            return (left + right) // 2

                    def CalculateMedian(numbers):
                        # 先把數字從小到大排序
                        numbers.sort()
                        length = len(numbers)

                        if length % 2 == 1:
                            # 奇數個數，直接取正中央那一個
                            return numbers[length // 2]
                        else:
                            # 偶數個數，取中間兩個數字的平均值
                            left = numbers[length // 2 - 1]
                            right = numbers[length // 2]
                            return (left + right) // 2

                    def CalculateMedian(numbers):
                        numbers.sort()
                        length = len(numbers)

                        if length % 2 == 1:
                            return numbers[length // 2]
                        else:
                            left = numbers[length // 2 - 1]
                            right = numbers[length // 2]
                            return (left + right) // 2

                    def main():
                        numbers = []
                        while True:

                                n = int(input())
