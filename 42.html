42. 快樂數


假設將一個正整數 $S_0$ 各個位數的平方和表示為 $S_1$。以此類推，將 $S_1$ 各個位數的平方和表示為 $S_2$，並持續重複這個過程。

如果對於某個 $i \ge 1$，其結果 $S_i = 1$，那麼原本的整數 $S_0$ 就被稱為「快樂數（Happy number）」。反之，不快樂的數字則被稱為「不快樂數（Unhappy number）」。

例如，7 是一個快樂數，因為它的變化過程為：
  7 $\rightarrow$ 49 $\rightarrow$ 97 $\rightarrow$ 130 $\rightarrow$ 10 $\rightarrow$ 1

而 4 是一個不快樂數，因為它的變化過程為：
  4 $\rightarrow$ 16 $\rightarrow$ 37 $\rightarrow$ 58 $\rightarrow$ 89 $\rightarrow$ 145 $\rightarrow$ 42 $\rightarrow$ 20 $\rightarrow$ 4（陷入迴圈）

輸入與輸出說明


輸入說明：

  輸入包含多組測試資料。輸入的第一行會給定測試資料的總數。
  每組測試資料僅佔一行，包含一個小於 $10^9$ 的正整數 $N$。

輸出說明：

  對於每組測試資料，你必須印出以下其中一種訊息：
    Case #p: N is a Happy number.
    Case #p: N is an Unhappy number.

  這裡的 p 代表測試資料的編號（從 1 開始計算）。如果數字 $N$ 是快樂數，你應該印出第一種訊息。否則，印出第二種訊息。

7
4
13 		Case #1: 7 is a Happy number.
Case #2: 4 is an Unhappy number.
Case #3: 13 is a Happy number.

                        import sys

                        def solve():
                            # 讀取所有輸入內容
                            input_data = sys.stdin.read().split()
                            if not input_data:
                                return
                                
                            T = int(input_data[0]) # 測試資料筆數
                            
                            for cas in range(1, T + 1):
                                original_n = int(input_data[cas])
                                n = original_n
                                seen = set() # 建立一個集合來記錄出現過的數字
                                
                                while n != 1:
                                    if n in seen:
                                        break # 數字重複出現，代表陷入迴圈
                                    seen.add(n)
                                    
                                    # 計算各位數的平方和 (Python 簡潔寫法)
                                    n = sum(int(digit)**2 for digit in str(n))
                                    
                                # 輸出結果
                                if n == 1:
                                    print(f"Case #{cas}: {original_n} is a Happy number.")
                                else:
                                    print(f"Case #{cas}: {original_n} is an Unhappy number.")
                        
                        if __name__ == '__main__':
                            solve()
