42. 快樂數


假設將一個正整數 $S_0$ 各個位數的平方和表示為 $S_1$。以此類推，將 $S_1$ 各個位數的平方和表示為 $S_2$，並持續重複這個過程。

如果對於某個 $i \ge 1$，其結果 $S_i = 1$，那麼原本的整數 $S_0$ 就被稱為「快樂數（Happy number）」。反之，不快樂的數字則被稱為「不快樂數（Unhappy number）」。

例如，7 是一個快樂數，因為它的變化過程為：
  7 $\rightarrow$ 49 $\rightarrow$ 97 $\rightarrow$ 130 $\rightarrow$ 10 $\rightarrow$ 1

而 4 是一個不快樂數，因為它的變化過程為：
  4 $\rightarrow$ 16 $\rightarrow$ 37 $\rightarrow$ 58 $\rightarrow$ 89 $\rightarrow$ 145 $\rightarrow$ 42 $\rightarrow$ 20 $\rightarrow$ 4（陷入迴圈）

輸入與輸出說明


輸入說明：

  輸入包含多組測試資料。輸入的第一行會給定測試資料的總數。
  每組測試資料僅佔一行，包含一個小於 $10^9$ 的正整數 $N$。

輸出說明：

  對於每組測試資料，你必須印出以下其中一種訊息：
    Case #p: N is a Happy number.
    Case #p: N is an Unhappy number.

  這裡的 p 代表測試資料的編號（從 1 開始計算）。如果數字 $N$ 是快樂數，你應該印出第一種訊息。否則，印出第二種訊息。

輸入輸出範例 Input Output
範例測資 3
7
4
13 		Case #1: 7 is a Happy number.
Case #2: 4 is an Unhappy number.
Case #3: 13 is a Happy number.

語法小教室：set() 集合


set() 就像是一個「不能放重複物品的百寶袋」。
在快樂數這題，我們用它來記錄數字變化時踩過的「足跡」。 


                    seen = set() # 建立一個空的集合
                    
                    seen.add(4)
                    seen.add(16)
                    seen.add(4)  # 試圖放入重複的數字會被自動無視！

                    print(seen)  # 結果會是 {4, 16}

                    # 檢查數字有沒有在裡面，速度極快！
                    if 16 in seen:
                        print("16 已經出現過囉！")

語法小教室：一行算完平方和


程式碼：n = sum(int(digit)**2 for digit in str(n))
這個寫法叫做「生成器運算式」，其實它就是把以下四行程式碼縮寫成一行： 


                    n = 49
                    total = 0

                    # str(n) 會把 49 變成字串 "49"，然後迴圈會分別拿出 '4' 和 '9'
                    for digit in str(n):
                        number = int(digit)    # 轉回整數 4 
                        total += number ** 2   # 計算 4 的平方(16) 並加到總和裡
                        
                    n = total # 最後 n 變成 97

解答

主程式 

					def solve():
						# 1. 讀取測試資料的總筆數 T
						T = int(input()) 
						
						# 準備一個空的收集箱(陣列)，用來存放所有的輸出結果
						results = []
						
						# 2. 從第 1 個數字開始，逐一處理每筆測試資料
						for cas in range(1, T + 1):
							n = int(input())
							original_n = n  # 把原始數字存起來，最後印出訊息時會用到
							
							seen = set() # 建立集合，記錄這個數字變化過程中出現過的值
							
							# 3. 當數字還沒變成 1 時，持續計算
							while n != 1:
								if n in seen:
									break # 如果數字重複出現了，代表陷入無窮迴圈，強制跳出
								
								seen.add(n) # 把當下的數字記錄到集合中
								
								# 把數字轉成字串拆開，算出每個位數的平方後相加
								n = sum(int(digit)**2 for digit in str(n))
								
							# 4. 判斷結果，並用 append() 塞進收集箱，先不印出來
							if n == 1:
								results.append(f"Case #{cas}: {original_n} is a Happy number.")
							else:
								results.append(f"Case #{cas}: {original_n} is an Unhappy number.")

						# 5. 所有數字都輸入且計算完畢後，把收集箱裡面的結果一次全部印出來
						for res in results:
							print(res)

					if __name__ == '__main__':
						solve()
